Paper Trading
Simulated bets using real Betway odds — no real money
Auto-trader running — bets placed 90 min before each game
How this works
  • Automatic: 90 minutes before each NHL game, the model runs, grabs real Betway odds, and places a paper bet if it finds value (edge > 3%).
  • Settlement: Each morning at 08:00, yesterday's game results are fetched from the NHL API and all pending bets are settled automatically.
  • Strategy: Use the dropdown below to change which signal filter the auto-trader uses. Click Save & Apply — new bets from that point forward will use the new strategy.
  • Manual triggers: Use the buttons below to place bets right now or settle immediately — useful for testing.
